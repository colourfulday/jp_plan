##### 好古園

姬路城附近，吐过愿意可以顺便逛一下

时间:  9時～17時（入園は16時30分まで

1.定移动wifi（有人说当地手机卡更好，没试过），下载google map，有道翻译官。
2.ICOCA
ICOCA+HARUKA套餐（单程即可，回程走关空快速），在国内时候多一个网上申请（最多提前一个月），到关西机场后直接在JR站里付钱取就行，很方便。原因是这次发现有ICOCA给行程带来很大的便利，强烈推荐。不搞套餐也要去搞卡，还不如直接一起了。
ICOCA类似交通一卡通，关西地区绝大部分交通工具可以使用（旅游要用到的，除了岚山小火车都包括了），全家、罗森、711三个当地主流便利店以及大部分商店都支持（在关东应该也可以用，和suica互通），省去了很多零钱的麻烦，非常实用。不买上面那个HARUKA套餐的话，也可以直接在各个地铁站、JR站的自助充值机那里买卡，但是稍微注意一下，除了押金之外，退卡时会收220yen的工本费。
3.其他交通问题
如果只在关西一地的话，不太建议买JR PASS，反过来讲如果需要坐2次及以上新干线的，可以考虑全国范围的JR PASS，这是从经济角度考虑的结果。比如关西的JR PASS，要想打平成本的话，基本上要买5天那个版本（否则每增加一天的边际成本比较高），同时还要多去关西各个城市（如奈良、神户、姬路，包括京都周围的岚山、伏见、宇治等等），总之就是短期不划算。
在有ICOCA的基础上，出行不太需要考虑太多，近铁和JR乘坐感差不多的，以便利为主。
关西地区比较推荐的交通优惠包括：京都的巴士1日券（500，京都市内基本靠巴士，坐3次就回本），大阪好像有个市内交通1日券（插一句，这种市内交通1日券一般都比较划算，做行程的时候稍微注意一下就好，东京也是，只是难度更高一些，因为交通运营公司更多）
4.行程及住宿
关西地区游一般两种住宿的思路，一种是先住京都后住大阪，因为购物主要在大阪比较方便，而且回程一般也会选在关西国际机场。另一种是都住在大阪，优点是遇到意外情况（如下雨）行程调整起来比较灵活，随时可以购物，不用拖着行李搬运（比如楼主的行程），缺点是交通上花的时间会更长一些。
总体来说推荐后者，但对第一次去关西的游客来说，前者也有好处，可以选择京都的民宿（和式风格，但较贵）体验一把。
京都的住宿，想要体验的话推荐祗园的民宿，否则京都站附近就可以了。大阪的话在梅田交通较为方便，天王寺的话离购物的心斋桥还是稍微有点距离的，但基本上选在难波也没什么问题。日本的宾馆（除了和式民宿）普遍房间小，但是东西很全。
P.S.如果关西机场出来直接去大阪梅田的话不能坐HARUKA。

5.饮食
在日本可以吃的很贵（好像是句废话），但是很难吃的很便宜。打预算的话，每人每顿正餐请至少预估在1200yen的水平，每人每天留出3000yen作为饮食预算（因为路上好吃的零食很多，加上买水什么的，可能还会超）。目前经历过吃的最便宜的，撇开便利店买面包这种，就是松屋、吉野家这些定食店了，牛肉饭都很不错，推荐，大概500yen一人。
此外，日本的蔬果普遍很贵，不推荐，看到过最便宜的应该是香蕉。

6.景点
最后扯扯关西的景点。
（1）京都：如果是第一次来日本的话，可以以传统文化游为主，金阁寺、银阁寺、平安神宫、下鸭神社、清水寺、祗园、二条城，再加个岚山就差不多了。这里面重点推荐清水寺及祗园一带，可以多逛逛，还有岚山，二条城也比较有趣，其他基本上和照片上看起来差不多，到一下就行。
（2）大阪：大阪主要是买买买……心斋桥道顿堀又吃又买，其他还有通天塔，大阪城，海游馆，这三个都还不错。逛街的话梅田一带选择也很多。
（3）奈良：相当推荐。前几年去的时候还要靠两条腿，今年去的时候有观光巴士了，很方便。时间够的话东大寺-春日大社一条线和唐招提寺一条线都可以去，重点推荐前一条线，从奈良公园过去都是鹿。
（4）其他还推荐姬路、神户（这俩可以凑一天，略赶）、伏见稻荷大社，远一点还可以去名古屋。
